MERGE PDFS WITH PDFTK
---------------------
IN ORDER TO COMBINE A NUMBER OF PDFS INTO ONE FILE, IT IS NECESSARY TO USE PDFTK, WHICH IS INSTALLED ON THE RICHMOND FACILITATOR MACHINE, AND IS CONFIGURED TO BE IN THE PATH STATEMENT, SO NO MATTER WHAT FOLDER YOU ARE WORKING IN, IT SHOULD WORK.

NOTE THAT PDFTK IS COMMAND LINE ONLY.  USAGE DETAILS:

TO CREATE A SINGLE PDF FROM MULTIPLE FILES:

C:\Temporary> pdftk file1.pdf file2.pdf cat output merged.pdf
