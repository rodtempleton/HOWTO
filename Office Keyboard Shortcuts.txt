Ctrl `				Toggle formulas on and off

Ctrl Shift Enter	Enter array formula
F9					Calculate worksheets
Shift F9			Calculate current worksheet
Ctrl Alt F9			Calculate all worksheets
Ctrl Shift U		Expand or collapse the formula bar
Ctrl A				Display function arguments dialog box
F11					Create default chart
Ctrl Shift +		Insert blank cells

Ctrl Shift Space	Select entire table
Ctrl ,				Copy cell value from cell above

Ctrl 1				Format cells dialog box
Tab					Next cell
F7					Spelling check
